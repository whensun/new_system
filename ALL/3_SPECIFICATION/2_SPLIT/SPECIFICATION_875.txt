## 21.2.6. Hypervisor Counter-Enable ( hcounteren ) Register

The counter-enable register hcounteren is a 32-bit register that controls the availability of the hardware performance monitoring counters to the guest virtual machine.

When the CY, TM, IR, or HPM\_n\_ bit in the hcounteren register is clear, attempts to read the cycle , time , instret ,  or hpmcounter n register while V=1 will cause a virtual-instruction exception if the same bit in mcounteren is 1. When one of these bits is set, access to the corresponding register is permitted when V=1,  unless  prevented  for  some  other  reason.  In  VU-mode,  a  counter  is  not  readable  unless  the applicable bits are set in both hcounteren and scounteren .
