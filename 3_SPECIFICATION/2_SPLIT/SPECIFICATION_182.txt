A 16-bit instruction with all bits zero is permanently reserved as an illegal instruction.

<image>

We reserve  all-zero  instructions  to  be  illegal  instructions  to  help  trap  attempts  to execute  zero-ed  or  non-existent  portions  of  the  memory  space.  The  all-zero  value should  not  be  redefined  in  any  non-standard  extension.  Similarly,  we  reserve instructions with all bits set to 1 (corresponding to very long instructions in the RISCV variable-length encoding scheme) as illegal to capture another common value seen in non-existent memory regions.
